# Directory Structure

## Root Level
- `src/` - Frontend source code (React components, hooks, pages, utils).
- `supabase/` - Backend code, including migrations, database schema, RPCs, and Edge Functions.
- `api/` - Auxiliary API routes/endpoints (potentially for webhooks or serverless handlers).
- `docs/` - Project documentation.
- `scripts/` - Scripts for maintenance or build steps.
- `public/` - Static assets.

## Key Files
- `package.json` - Node dependencies and scripts.
- `PROJECT_CONTEXT.md` - High-level overview of project goals, architecture, and critical business rules.
- `TECHNICAL_GUIDE.md` - Detailed technical documentation.
- `DOCUMENTATION_INDEX.md` - Index for project documentation.
- `tailwind.config.ts`, `vite.config.ts` - Configuration files for styling and build process.
