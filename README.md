## 🎯 Visão Geral

Este sistema foi desenvolvido para **automatizar e otimizar todo o processo de captação de novos alunos** em escolas e instituições de ensino. Desde a inscrição inicial do candidato até sua matrícula final, o sistema oferece ferramentas completas para:

- 📝 **Gerenciar inscrições online** de forma organizada
- 📅 **Agendar e controlar entrevistas** com candidatos
- 💰 **Calcular valores de mensalidades e materiais** automaticamente
- 📊 **Gerar relatórios e estatísticas** em tempo real
- 👥 **Controlar acesso de diferentes usuários** com permissões específicas
- 🔔 **Acompanhar o status** de cada candidato no processo

## ⭐ Principais Funcionalidades

### 📝 1. Sistema de Inscrições Online

**O que faz:** Permite que candidatos se inscrevam pela internet de forma simples e rápida.

**Como funciona:**
- Formulário online acessível por link único de cada unidade
- Coleta dados pessoais, acadêmicos e de contato
- Validação automática dos dados inseridos
- Confirmação por email após inscrição
- Permite múltiplos telefones de contato

**Benefícios:**
- ✅ Reduz trabalho manual da equipe
- ✅ Diminui erros de digitação
- ✅ Candidatos podem se inscrever 24/7
- ✅ Dados organizados desde o início

---

### 📅 2. Gestão de Agendamentos

**O que faz:** Controla toda a agenda de entrevistas e atendimentos aos candidatos.

**Como funciona:**
- Calendário visual com disponibilidade de horários
- Agendamento de entrevistas com entrevistadores específicos
- Notificações automáticas
- Controle de presença/ausência
- Reagendamento facilitado
- Opção de atendimento presencial ou à distância

**Benefícios:**
- ✅ Evita conflitos de horários
- ✅ Otimiza o tempo dos entrevistadores
- ✅ Reduz faltas com lembretes
- ✅ Histórico completo de agendamentos

---

### 💰 3. Cálculo Automático de Valores

**O que faz:** Calcula automaticamente mensalidades, descontos e valores de materiais didáticos.

**Como funciona:**
- Definição de valores por turma/série
- Aplicação de descontos percentuais
- Cálculo de material didático (anual ou mensal)
- Simulação de parcelamento
- Diferentes formas de pagamento (à vista, cartão, boleto)
- Geração de proposta financeira

**Benefícios:**
- ✅ Elimina erros de cálculo
- ✅ Apresentação profissional dos valores
- ✅ Transparência com os responsáveis
- ✅ Agilidade no atendimento

---

### 👤 4. Gestão Completa de Candidatos

**O que faz:** Acompanha cada candidato em todas as etapas do processo.

**Como funciona:**
- **Status automático:** Sistema atualiza status conforme ações
  - Não Confirmado
  - Confirmado
  - Atendimento Agendado
  - Atendimento Realizado
  - Matriculado
  - Desistente
  - Entre outros

- **Ficha completa do aluno:**
  - Dados pessoais
  - Informações acadêmicas
  - Histórico de interações
  - Notas de entrevista
  - Proposta financeira
  - Telefones de contato

- **Interações registradas:**
  - Comentários da equipe
  - Mudanças de status
  - Reagendamentos
  - Observações importantes

**Benefícios:**
- ✅ Visão 360° do candidato
- ✅ Histórico completo preservado
- ✅ Comunicação centralizada
- ✅ Facilita follow-up

## 📖 Guia de Uso

### **Para Administradores**

#### 1️⃣ **Configuração Inicial**
1. Acesse o sistema com credenciais de admin
2. Vá em **Configurações → Unidades**
3. Cadastre suas unidades escolares
4. Configure turmas e valores em **Configurações → Turmas**
5. Cadastre datas de provas em **Configurações → Datas de Provas**
6. Crie usuários da equipe em **Configurações → Usuários**

#### 2️⃣ **Compartilhamento do Link de Inscrição**
- Copie a URL: `https://seudominio.com/inscricao/slug-da-unidade`
- Divulgue nas redes sociais, site, WhatsApp, etc.

#### 3️⃣ **Monitoramento**
- Acesse o **Dashboard** para visão geral
- Use a aba **Relatórios** para análises detalhadas
- Exporte dados quando necessário

---

### **Para Entrevistadores**

#### 1️⃣ **Configurar Disponibilidade**
1. Acesse **Configurações → Disponibilidade**
2. Defina seus horários disponíveis
3. Salve as configurações

#### 2️⃣ **Gerenciar Agendamentos**
1. Vá em **Agendamentos**
2. Visualize o calendário com seus horários
3. Veja detalhes dos candidatos agendados

#### 3️⃣ **Realizar Atendimento**
1. Abra a ficha do candidato
2. Registre informações da entrevista
3. Adicione notas se necessário
4. Defina desconto (se aplicável)
5. Configure forma de pagamento do material
6. Clique em **Registrar Atendimento**
7. Sistema atualiza status automaticamente

---

### **Para Equipe Administrativa**

#### 1️⃣ **Gerenciar Inscrições**
1. Acesse a aba **Inscritos**
2. Busque candidatos por nome, código, telefone
3. Filtre por status, série, unidade
4. Abra a ficha para ver detalhes completos

#### 2️⃣ **Atualizar Status**
1. Abra a ficha do candidato
2. Selecione novo status
3. Adicione comentário (se necessário)
4. Confirme a atualização

#### 3️⃣ **Agendar Entrevistas**
1. Abra a ficha do candidato
2. Role até "Agendar Entrevista"
3. Selecione data, horário e entrevistador
4. Confirme o agendamento

---

### **Para Candidatos (Processo de Inscrição)**

#### Passo a Passo da Inscrição:

1. **Acesse o link** fornecido pela escola
2. **Dados do Aluno:**
   - Nome completo
   - Data de nascimento
   - CPF (opcional)

3. **Dados do Responsável:**
   - Nome
   - CPF
   - RG
   - Telefones de contato
   - Email

4. **Endereço:**
   - Cidade
   - Bairro
   - CEP (opcional)

5. **Dados Acadêmicos:**
   - Série pretendida
   - Turma
   - Escola de origem

6. **Informações Adicionais:**
   - Data de prova (se houver)
   - Como conheceu a escola

7. **Revisão e Confirmação**

8. **Receba confirmação** com código de inscrição

## 🚀 Instalação e Configuração

### **Pré-requisitos**
- Node.js 18+ instalado
- Conta no Supabase (gratuita)
- Git instalado

### **Passo 1: Clonar o Repositório**
```bash
git clone <url-do-repositorio>
cd aluno-first-crm
```

### **Passo 2: Instalar Dependências**
```bash
npm install
```

### **Passo 3: Configurar Variáveis de Ambiente**

Crie um arquivo `.env` na raiz do projeto:

```env
VITE_SUPABASE_URL=sua_url_do_supabase
VITE_SUPABASE_ANON_KEY=sua_chave_anonima
```

### **Passo 4: Configurar Banco de Dados**

1. Acesse o Supabase Dashboard
2. Crie um novo projeto
3. Execute as migrations em `supabase/migrations/`
4. Configure as políticas RLS

### **Passo 5: Executar em Desenvolvimento**
```bash
npm run dev
```

Acesse: `http://localhost:5173`

### **Passo 6: Build para Produção**
```bash
npm run build
```

Os arquivos otimizados estarão em `dist/`
